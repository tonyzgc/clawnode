Features

Single Purpose

One function: press to speak. Full-disc pressure key with tactile confirmation. Release to send, no second confirmation needed.

Directional Audio

Dual MEMS microphone array with beamforming. Optimized range: 60-100cm for desktop use. Activates only when pressed, immune to ambient noise.

Zero Config

USB-C wired connection, driver-free. Auto-detect on plug, no pairing required. Firmware OTA updates.

Use Cases

Dictation Writing
