# ClawNode

温润极简的 OpenClaw 语音终端。

**按下。说话。Claw 听见。**

ClawNode 是一款直径 60mm 的精密硬件设备，置于键盘左侧。按下即说，无需唤醒词，没有屏幕干扰 —— 只是桌面上的一个静谧存在。

## 设计理念

温润、极简、精致。如一块静置桌面的美玉。

- **瓷白质感** — 柔和哑光陶瓷涂层，温润触感
- **无缝形态** — 无可见接缝，无锐角
- **柔光指示** — 待机时柔和暖光，激活时温润白光
- **静谧存在** — 与工作空间自然融合

## 规格

- **尺寸**：⌀60mm × 10-15mm
- **重量**：约 120g
- **材质**：CNC 铝合金，陶瓷涂层
- **颜色**：瓷白（暖白）/ 砂岩（暖灰）
- **连接**：USB-C 2.0（1.5m 编织线）
- **麦克风**：双 MEMS 阵列，60-100cm 收音距离

## 功能

- **单点专注**：一个按键，一个功能 —— 按下说话
- **定向收音**：双 MEMS 麦克风阵列，波束成形
- **零配置**：即插即用，免驱
- **状态灯光**：极简 LED 指示（待机暖光，激活柔白）
