# ClawNode

温润极简的 OpenClaw 语音终端。

**按下。说话。Claw 听见。**

一枚静置于键盘旁的圆盘。无可见麦克风，无接缝——只有纯粹的温润陶瓷。按下即说。

## 设计

**无缝形态**
⌀60mm 无间断表面。麦克风隐藏于下方。无可见开孔，无边缘，无干扰。如一块光滑的玉石融入桌面。

**温润触感**
陶瓷涂层。如阳光温暖过的石头般触感。120g 令人安心的存在。

**柔光指示**
待机时温暖的呼吸。聆听时柔软的白光绽放。从不打扰，始终安心。

## 规格

- **尺寸**：⌀60mm × 12mm
- **重量**：约 120g
- **材质**：CNC 铝合金，陶瓷涂层
- **颜色**：瓷白（暖白）· 砂岩（暖灰）
- **连接**：USB-C（1.5m 编织线）
- **音频**：双 MEMS 麦克风，60-100cm 距离，隐藏于表面之下

## 使用方法

1. 放于键盘左侧
2. 插入 USB-C
3. 按下说话
