# ClawNode

OpenClaw 桌面语音终端。

**按下。说话。Claw 听见。**

ClawNode 是一款直径 60mm 的精密硬件设备，置于键盘左侧。按下即说，无需唤醒词，没有屏幕干扰，纯粹的物理交互。

## 概述

- **尺寸**：⌀60mm × 10-15mm
- **重量**：约 120g
- **材质**：CNC 铝合金一体成型，阳极氧化处理
- **连接**：USB-C 2.0（1.5m L型线）
- **麦克风**：双 MEMS 阵列，60-100cm 收音距离

## 功能

- **单点专注**：一个按键，一个功能 —— 按下说话
- **定向收音**：双 MEMS 麦克风阵列，波束成形
- **零配置**：即插即用，免驱
- **状态灯光**：极简 LED 指示（待机微光蓝，激活白光）
